## 🎯 What it does

ResumeForge AI takes your existing resume and a target job description, then uses AI to generate a customized resume optimized for that specific role:

- **ATS keyword matching** — Ensures your resume passes Applicant Tracking System filters
- **Smart rewriting** — Highlights and restructures your most relevant experience
- **Quantified achievements** — Transforms vague descriptions into impactful, measurable statements
- **Instant results** — 10 seconds from paste to polished resume

## 🚀 Quick Start

1. **Visit** [ResumeForge AI](https://tyr1105.github.io/resumeforge-ai/)
2. **Get a free API key** from [DeepSeek](https://platform.deepseek.com/api_keys) (free credits available)
3. **Paste** the job description you're applying to
4. **Paste** your current resume
5. **Click** "Generate" → Get your tailored, ATS-optimized resume!

## 🏗️ Tech Stack

- **Pure HTML/CSS/JS** — No frameworks, no dependencies
- **Static site** — Hosted on GitHub Pages
- **No backend** — API calls go directly from your browser to DeepSeek
- **No database** — Zero server-side data storage
- **localStorage** — API key stored locally, never uploaded

## 🔒 Privacy

| Data | Where it goes |
|------|--------------|
| Your resume | Sent only to DeepSeek API (your key, your account) |
| Your API key | Stored in your browser's localStorage only |
| Usage tracking | Client-side only (count stored locally) |
| Server logs | None (static site, no backend) |
